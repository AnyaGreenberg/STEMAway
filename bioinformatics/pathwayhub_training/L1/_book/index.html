Level 1: Bioinformatics workflow applied to transcriptomics

Level 1: Bioinformatics workflow applied to transcriptomics

Anya Greenberg

2021-08-17

Project definition

Recommended pre-requisites

  • Basic understanding of genetics
  • Beginner experience with R, Python, or an equivalent statistical language

Technical requirements

  • 8 GB RAM
  • 5 GB free disk space

If you don’t meet the requirements above, contact @anya on the STEM-Away forums.

Objectives

  1. Learn about the important concepts of bioinformatics spanning computer science, statistics, and biology.
  2. Understand a generalized workflow for transcriptomic analysis of microarray data.
  3. Implement workflow in intern-led research project.

Technical skills objectives

  • Learn about important concepts of bioinformatics
    • Computer science
      • Data structures
      • Data analysis and visualization
      • Functions and abstraction
    • Introduction to statistics for bioinformatics
      • Hypothesis testing
      • Clustering and similariyt metrics
    • Introduction to transcriptomics
  • Application of generalized workflow for transcriptomic analysis
    • Data acquisition from NCBI’s Gene Expression Omnibus (GEO)
    • Quality control - inter-array vs. intra-array
    • Differential gene expression (using linear modeling)
    • Computational functional analysis
      • Enrichment
      • Over-representation
      • Pathway/ networks
  • Use R programming and other bioinformatic tools in transcriptomic research
    • R syntax, pacakges, documentation, and troubleshooting
    • Database for Annotation, Visualization, and Integrated Discover (DAVID)
    • Enrichr
    • Metascape: a gene anotation and analysis resource
    • STRING: functional protein association networks
    • Gene Expression Profiling Interactive Analysis (GEPIA)
  • Research practices
    • Critical thinking
    • Literature searches
    • Telling a story

Soft skills objectives

  • Project and time management
  • Virtual collaboration and teamwork
  • Technical and project presentations
  • Leadership and mentoriing (student leads)
